## 🤝 איך להצטרף לרשימה?

תרומות לפרויקט SoundFlow הן תמיד מבורכות! אם תרמת לפרויקט, תוכל להוסיף את עצמך לרשימה.

### איך לתרום?

1. **Fork** את הפרויקט
2. צור **branch** חדש (`git checkout -b feature/AmazingFeature`)
3. בצע **commit** לשינויים (`git commit -m 'Add some AmazingFeature'`)
4. **Push** ל-branch (`git push origin feature/AmazingFeature`)
5. פתח **Pull Request**

### סוגי תרומות מבורכות:

- 🐛 דיווח על באגים
- 💡 הצעות לשיפורים
- 📝 שיפור תיעוד
- 🎨 שיפורי עיצוב
- 🧪 הוספת בדיקות
- 🌐 תרגומים
- 🎵 הוספת כלי נגינה, סגנונות או מצבי רוח
- 🎬 יצירת שירים נוספים
- ⚡ שיפורי ביצועים
- 🔒 שיפורי אבטחה
