# SoundFlow 🎵

אתר סטטי ליצירת הוראות מדויקות (prompts) למוזיקה עם Stable Audio.

## 🏛️ החזון

**SoundFlow** הוא חלק מחזון גדול יותר: **סט כלים למנצח התזמורת בבית המקדש שיבנה במהרה בימינו בעזרת השם!**

כל שיר, כל כלי נגינה, כל סגנון - הכל נוצר כדי להיות חלק מהתזמורת הגדולה שתיצור את המוזיקה בבית המקדש.

📖 [קרא עוד על החזון](VISION.md)

## מבנה הפרויקט

```
SoundFlow/
├── index.html          # עמוד ראשי
├── styles.css          # עיצוב
├── app.js              # לוגיקה ראשית
├── data/
│   ├── instruments.json    # כלי נגינה
│   ├── styles.json         # סגנונות מוזיקליים
│   └── moods.json          # מצבי רוח
└── README.md
```

## שימוש

1. פתח את `index.html` בדפדפן
2. בחר כלי נגינה (תיבות סימון)
3. בחר סגנון מוזיקלי
4. בחר מצב רוח
5. הגדר קצב (BPM)
6. הוסף מילות מפתח (אופציונלי)
7. הזן כותרת מותאמת אישית (אופציונלי)
8. לחץ על "צור פרומפט"
9. העתק או הורד את הפרומפט שנוצר

## תכונות

- ✅ רשת של כלי נגינה עם תיבות סימון
- ✅ בחירת סגנון מוזיקלי
- ✅ בחירת מצב רוח
- ✅ בחירת קצב פעימות (BPM) עם slider ו-presets
- ✅ הוספת מילות מפתח מותאמות אישית
- ✅ יצירת פרומפט אוטומטי
- ✅ העתקה ל-clipboard
- ✅ הורדה כקובץ טקסט
- ✅ עיצוב מודרני ויפה
- ✅ רספונסיבי למובייל

## טכנולוגיות

- HTML5
- CSS3 (עם Gradients ו-Backdrop Filters)
- JavaScript (Vanilla)
- JSON לנתונים
